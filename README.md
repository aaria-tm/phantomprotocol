# phantomprotocol
Core Features
Heuristic Analysis Engine: Performs real-time validation of top-level domains, high-risk urgency keywords, and email sender signatures.

Interactive Asset Scanner: Simulates a deep system scan using fluid, text-based status updates and custom progress trackers.

Actionable Security Verdicts: Delivers explicit safety markers like "Safe to use" or "Safe to enter information" without convoluted technical jargon.

Premium Micro-interactions: Built-in magnetic button physics that respond gracefully to cursor proximity without introducing jerky layout shifts.

Single-File Architecture: Zero dependencies, zero external libraries, and fully self-contained within an efficient HTML/CSS/JS file.

Design System
The application utilizes a refined, luxury-inspired digital layout designed for optimal scannability and visual balance:

Background Base: #e9e4d4 (Alabaster Beige)

Primary Text & Accents: #5a3e2b (Deep Espresso)

Safe State Border: #4a6b52 (Muted Olive)

Threat State Border: #8c3b2b (Crimson Earth)

Typography: Playfair Display (Serif Headers) paired with Inter (Clean Sans-Serif Body)

Technical Architecture
The architecture relies entirely on native browser APIs to maintain an ultra-lightweight performance profile.

State Machine Management
Transitions between the Entry Screen (Phase 1), the Running Scan Loop (Phase 2), and the Dynamic Verdict Display (Phase 3) are handled via a centralized JavaScript state engine (appEngine). This prevents redundant DOM queries and ensures instant layout rendering.

Installation and Deployment
Because the entire application is written as a unified script, setup requires no package managers or server environments.

Clone or download the repository files.

Locate the core file: phantom.html.

Double-click the file to launch the console directly in any modern web browser (Chrome, Safari, Edge, Firefox).
